scoverage:integration-test

Full name:

org.scoverage:scoverage-maven-plugin:1.4.0-M5:integration-test

Description:

Executes forked scoverage life cycle up to vefiry phase.

In forked scoverage life cycle project is compiled with SCoverage instrumentation and unit and integration tests are executed.

Attributes:

  • Requires a Maven project to be executed.
  • Since version: 1.2.0.
  • Invokes the execution of the lifecycle phase verify prior to executing itself.
  • Executes in its own lifecycle: scoverage.

Optional Parameters

Name Type Since Description
<skip> boolean 1.0.0 Allows SCoverage to be skipped.

Default value is: false.
User property is: scoverage.skip.
